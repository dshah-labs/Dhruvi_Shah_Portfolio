# Dhruvi Shah Portfolio

Personal portfolio site for Dhruvi Shah, focused on Data Science, AI Engineering, and ML systems work.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS (via CDN styles in `index.html`)
- Lucide Icons

## Local Development

Prerequisite: Node.js 18+

1. Install dependencies:
   `npm install`
2. Start local server:
   `npm run dev`
3. Open:
   `http://localhost:3000`

## Production Build

1. Build:
   `npm run build`
2. Preview:
   `npm run preview`

## Project Structure

- `App.tsx`: Main page layout and sections
- `constants.tsx`: Experience, projects, and skills content
- `components/AIChat.tsx`: AI chat widget
- `index.tsx`: React entry point
- `index.html`: Root HTML and global styles

## Notes

- Profile photo path used in hero section: `/public/profile.jpeg`
- GitHub links for featured projects are configured in `constants.tsx`
